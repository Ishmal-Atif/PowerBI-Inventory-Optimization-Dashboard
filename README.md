# PowerBI-Inventory-Optimization-Dashboard
Inventory optimization and performance monitoring dashboard built in Power BI.

## Project Overview

This Power BI dashboard was developed to monitor inventory performance, demand forecasting accuracy, stock value distribution, and sales trends across multiple product categories and regions.

The dashboard supports data-driven inventory planning and operational decision-making.

## Tools Used

- Power BI
- DAX
- Power Query

## Business Objectives

- Monitor inventory value
- Evaluate inventory turnover performance
- Compare forecasted demand against actual sales
- Track sales trends over time
- Analyze inventory distribution across regions and categories

## Dashboard Features

### Inventory KPIs
- Total Stock Value
- Inventory Turnover
- Competitor Price Index

### Inventory Analysis
- Inventory Level by Region
- Inventory Level by Product Category

### Demand Forecasting
- Forecast vs Actual Analysis
- Product Category Performance

### Sales Monitoring
- Historical Sales Trends
- Regional Performance Tracking

### Interactive Filters
- Month Selection
- Region Selection
- Product Category Selection

## Key Insights

- Inventory value exceeded $1 billion across monitored categories.
- Inventory turnover metrics helped identify operational efficiency.
- Forecasted demand closely aligned with actual sales across multiple categories.
- Regional inventory distribution remained relatively balanced.
- Sales trends highlighted periods of stronger and weaker demand.
